# 特别兴趣小组

为了增强社区成员之间的透明度和协作，TuGraph 创建了一系列特别兴趣小组（SIGs）。

# SIG 列表

当前已创建的 SIG 有：

| SIG      | Repo                                           | Description                     |
|----------|------------------------------------------------|---------------------------------|
| GraphRAG | [DB-GPT][repo1]                                | 基于 DB-GPT 的 TuGraph GraphRAG 框架 | 
| OSGraph  | [OSGraph][repo2]                               | 基于 TuGraph 的开源数据洞察              |
| Text2GQL | [Awesome-Text2GQL][repo3], [DB-GPT-Hub][repo4] | Text2GQL 语料生成与微调                |
| mini-gu  | [miniGU-design-doc][repo5]                     | 面向科研的轻量级图数据库                    |
