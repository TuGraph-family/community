# Special Interest Groups

To enhance community transparency and collaboration among community members, 
TuGraph created a series of Special Interest Groups (SIGs).

# SIG list

The currently created SIGs are:

| SIG      | Repo                                           | Description                                |
|----------|------------------------------------------------|--------------------------------------------|
| GraphRAG | [DB-GPT][repo1]                                | TuGraph GraphRAG Framework on DB-GPT       | 
| OSGraph  | [OSGraph][repo2]                               | Open-source Data Insight on TuGraph        |
| Text2GQL | [Awesome-Text2GQL][repo3], [DB-GPT-Hub][repo4] | Text2GQL Corpus Generation and Fine Tuning |
| mini-gu  | [miniGU-design-doc][repo5]                     | Lightweight Graph Database for Research    |
